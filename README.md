# Delegate - AI Automation for UK SMEs

Official website for **Delegate** - done-for-you AI automation, built, deployed, and managed for UK businesses.

**Live site:** [delegate-me.com](https://delegate-me.com)

## About Delegate

We help UK SMEs automate repetitive tasks with AI, freeing up time to focus on growth. Our solutions include:

- **AI Proposal Generator** - Create professional proposals in minutes
- **Knowledge Chatbot** - Instant access to company knowledge
- **Lead Scoring System** - Automated prospect discovery and qualification
- **Status Reports** - Auto-generated project updates
- **Expense Categorization** - AI-powered bookkeeping automation

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **3D Graphics:** Three.js / Spline
- **Language:** TypeScript

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
